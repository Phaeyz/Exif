# TagProvider.AddOrUpdate method

Adds a new tag to the provider. If an equivalent tag already exists, it will be replaced. The tag's related and parents, if available, will also be added or updated.

```csharp
public bool AddOrUpdate(Tag tag)
```

| parameter | description |
| --- | --- |
| tag | The tag to add to the provider. |

## Return Value

Returns true if the tag previously did not exist and now it exists in the provider, or false otherwise.
