# ImageFileDirectoryCollection.Deserialize method

Deserializes a buffer as EXIF image file directory collection.

```csharp
public static ImageFileDirectoryCollection Deserialize(ReadOnlyMemory<byte> exifBuffer, 
    int startingOffset, ByteOrderMode byteOrderMode, TagProvider? tagProvider, 
    out List<EntryReference> cannotRoundTripEntries)
```

| parameter | description |
| --- | --- |
| exifBuffer | The buffer containing the EXIF image file directory collection. |
| startingOffset | The offset within the EXIF buffer where the image file directory collection begins. |
| byteOrderMode | The byte order mode of the data serialized in the buffer. |
| tagProvider | A provider used for sourcing tag details required for deserialization instructions. If null, the default tag provider is used. |
| cannotRoundTripEntries | On output, this receives a list of entries which, without intervention, may not round trip. This means if the image file directory collection is reserialized, these entries may not be valid. |

## Return Value

Returns the deserialized EXIF image file directory collection.
