Creates an offset and length pair tag.

```csharp
public static Tag CreateOffsetAndLengthPair(Tag? parent, int? index, ushort offsetValue, 
    string? offsetName, ushort lengthValue, string? lengthName, 
    IEnumerable<(string key, string category)>? aliases)
```

| parameter | description |
| --- | --- |
| parent | The parent directory tag the current tag must exist under. If `null` this tag may appear in every parent directory. |
| index | The index of parent directory set which this tag must exist within. If `null` this tag may appear in any of the array of parent directories. |
| offsetValue | The UINT16 value of the tag for the data offset. |
| offsetName | A friendly name for the offset tag. Only used for debugging. |
| lengthValue | The UINT16 value of the tag for the data length. |
| lengthName | A friendly name for the length tag. Only used for debugging. |
| aliases | Aliases for the tag. The key a namespace name such as "Exif.Image.ImageWidth", and the value is the category such as "Image". |
