# Tag.CreateIfdPointer method (1 of 2)

Creates a directory pointer tag.

```csharp
public static Tag CreateIfdPointer(Tag? parent, int? index, ushort value, string? name, 
    bool scopeByteOffset, IEnumerable<(string key, string category)>? aliases)
```

| parameter | description |
| --- | --- |
| parent | The parent directory tag the current tag must exist under. |
| index | The index of parent directory set which this tag must exist within. If `null` this tag may appear in any of the array of parent directories. |
| value | The UINT16 value of the tag which is serialized to the EXIF. |
| name | A friendly name for the tag. Only used for debugging. |
| scopeByteOffset | If true, all offsets within the serialized image file directory are relative to the first byte of the image file directory. If false, all offsets are relative to the start of the EXIF buffer (this is the normal case). |
| aliases | Aliases for the tag. The key a namespace name such as "Exif.Image.ImageWidth", and the value is the category such as "Image". |

## Return Value

The new directory pointer tag.
